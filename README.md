# УмныйКласс — маркетинговый лендинг (Next.js)

Фронтенд-проект для дипломной темы: AI-платформа онлайн-репетиторства с живыми уроками (видеосвязь + инструменты класса).

## Стек

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS

## Установка

```bash
npm install
```

## Запуск в разработке

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
```

## Прод-старт

```bash
npm run start
```

## Маршруты

- `/` — лендинг (одностраничная структура с якорями)
- `/login` — заглушка входа
- `/signup?role=student|tutor` — заглушка регистрации с ролью
- `/lead?role=student|tutor` — заявка на консультацию/созвон (без обязательной регистрации)
- `/assessment` — предварительное тестирование для определения уровня знаний
- `/teachers` — каталог преподавателей (поиск/фильтры/сортировка)
- `/teachers/[id]` — детальный профиль преподавателя (вкладки, расписание, отзывы, карточка бронирования)
- `/for-tutors` — страница для привлечения преподавателей
- `/app` — редирект в кабинет ученика `/app/dashboard`
- `/app/dashboard` — дашборд ученика (KPI, активные курсы, календарь, домашка, ИИ-инсайты)
- `/app/courses` — каталог курсов с фильтрами/поиском/категориями
- `/app/teachers` — каталог преподавателей внутри личного кабинета
- `/app/teachers/[id]` — профиль преподавателя в интерфейсе кабинета
- `/app/lessons` — расписание занятий с календарем и фильтрацией
- `/app/lessons/[id]` — детали занятия (конспект, домашка, словарь, файлы)
- `/app/homework` — список домашних заданий с фильтрами
- `/app/vocabulary` — словарь ученика (поиск, теги, избранное)
- `/app/messages` — сообщения с преподавателями (UI-заглушка)
- `/app/analytics` — отдельная страница аналитики (динамика, цели, предметы, зоны внимания)
- `/app/payments` — история платежей и пакеты занятий (UI-заглушка)
- `/app/settings` — профиль и настройки ученика
